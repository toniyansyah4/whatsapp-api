## Setup
- `cp .env.example .env`
- Now change `.env` file
- Change configuration DB and Pusher
- Run `composer install`
- Run `php artisan migrate`
- Run `npm install`
- Run `npm run dev`
- Run `php artisan db:seed`
- Run `php artisan key:generate`
- Run `php artisan storage:link`
- Run `php artisan serve`
